# Run an Arbitrum Orbit node

<Callout type="info" emoji="ℹ️">
  The external nodes feature must be enabled on the network for this to work. If
  you are the owner of a network, you can enable this feature from your
  deployment settings in the Conduit app.
</Callout>

To run a node for an Arbitrum Orbit rollup deployed on Conduit, please follow [Arbitrum's documentation](https://docs.arbitrum.io/node-running/how-tos/running-an-orbit-node).

The sequencer feed input url will be `https://relay-[your_conduit_slug_here].t.conduit.xyz`
